### Summary

### Setup

* npm install

* .env : DATABASE_URL: user, password, host, portNo. dbname
```
DATABASE_URL="mysql://user1234:password@host1234:3306/dbname"
```

* apollo-client.ts

uri: if change domain, setting require
```
uri: 'http://localhost:3000/api/graphql',
```

***
### Start server
* start :

yarn dev
